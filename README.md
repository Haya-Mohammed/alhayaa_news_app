# alhayaa_news_app

A News app

## Tools: 
- Splash Screen (with flutter_native_splash)
- RESTful API (newsapi.com)
- BLoC (with Cubit)
- MVVM Pattern
- Chache images (with cached_network_image)
- Dark Theme (with Shared Preference)
- Hive (For storing articles locally)
- Font Added
- App icon Added
